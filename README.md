## About This Project
I wanted to move away from simple lists of links and create something that reflected my interest in technical systems. I chose a "Terminal" theme because it represents the tools I use every day while learning to code.

### My Goals for this Portfolio:
- **Honest Progress:** I’ve included skill bars (30-45%) to show exactly where I am in my learning process. I am focused on building a solid foundation in Java and Python.
- **Learning Web Dev:** This site was a way for me to practice my Frontend skills (HTML, CSS, and Vanilla JavaScript) alongside my Data Science studies.
- **Documentation:** It serves as a central hub for my school and personal projects.
